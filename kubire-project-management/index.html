01 / 前提整理

2つのプロジェクトを分けて管理する

今回の実務は、一つのプロジェクトではなく、短期売上を守る運用案件と、中期の収益構造を作る制作案件の2本立てです。

Project A

既存の仕組み（IG主体の集客）

  • 今のフォロワー、DM、問い合わせから講座を埋める
  • 過去にどの訴求でどれだけ集まったかを定量化する
  • 価格で止まった人、断った人の理由も整理する
  • 質を落とさず、利益率を下げずに埋める設計が必要
Project B

コンテンツマーケの仕組み

  • 無料動画や説明会で教育してからセールスする導線
  • 久保さん自身はコンテンツホルダーに寄せる
  • 司会、個別相談、セールスは別人材前提
  • セールスフィーを払う座組も設計対象に入れる

07 / 見るべきKPI

運用で追うべき数字

既存運用と新導線で追う数字は分けておいた方が判断しやすいです。

既存講座運用KPI

  • フォロワー増加数
  • DM数、問い合わせ数
  • 申込率
  • 価格で離脱した件数
  • 開催日ごとの申込人数

コンテンツ導線KPI

  • オプトイン数
  • 動画視聴完了率
  • 説明会申込率 / 購入率
  • 個別相談予約率
  • 個別相談成約率
管理のコツ: まずは Project A で現状の数字を定量化しながら、Project B を制作進行するのがよいです。いきなり新導線だけに振るより、既存講座の集客実績がそのまま設計のヒントになります。
