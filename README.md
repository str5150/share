# Share

Cursor で作成したコンテンツの公開リポジトリ。GitHub Pages で配信。

## 構造

```
share/
├── index.html              ← ルートページ（コンテンツ一覧）
├── omoya-training/         ← Claude Code 導入講座
│   ├── index.html
│   ├── curriculum/
│   └── assets/
└── (next-project)/         ← 次のコンテンツをここに追加
```

## コンテンツの追加方法

1. `share/` 直下に新しいフォルダを作成
2. HTML ファイルを配置
3. `index.html` にカードを追加
4. push すれば自動で公開
