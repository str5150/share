01

薄毛男性のリアルな心理構造

選択肢の限界を知っている

  • AGAや薬は検討済み。でも価格が高い、飲み続けないといけない
  • 金髪坊主やスキンヘッドは仕事上無理
  • カットだけで全体を隠すのは限界がある。でもできるだけ隠したい
  • 「魔法の解決策」を求めてはいない。現実的な範囲での最善策を探している

美容室選びは命がけ

  • 一番怖いのは新しい美容室に行って薄毛が目立つカットにされること
  • 今の美容師に薄毛の悩みを伝えて、ちゃんとやってくれるなら一生離れない
  • 新しい場所でまた一から説明するのが面倒だし、失敗リスクが高すぎる
  • スイッチングコストが異常に高い。一度獲得すれば月イチで一生通う

「薄毛専門」の看板は諸刃の剣

  • スヴェンソンのような明らかに薄毛サロンだとわかる場所には行きたくない
  • 周囲に「あいつ薄毛サロン行ってるぞ」と思われたくない
  • サルタヒコはここのバランスが絶妙。薄毛専門だけど普通の美容室に見える

求めているのは「職人」

  • 薄毛に合わせたカットはほぼ職人芸。刈り上げ、ツーブロック、グラデーションの微調整
  • 頭頂部が薄い人は前髪・後ろ髪との長さの繋がりが命
  • M字の人はサイドの処理が全て
  • この微妙な差を理解している美容師はごくわずか
  • SNSで「この人、わかってる」と思わせたら勝ち

03

コンテンツ戦略

最優先：薄毛タイプ別の「技術力」発信

これが一番効く。薄毛男性は自分のタイプ（M字、頭頂部、U字、全体的に薄い等）を正確に認識している。自分と同じタイプの人がどう整えられているかを見たときに「この人、わかってる」と感じる。

POST EXAMPLES
  • 「M字が進行してきた方。サイドを3mmで刈り上げて、前髪は立たせずに流す。これだけでM字のラインが目立たなくなる」→ 具体的なミリ数まで言い切る
  • 「頭頂部が薄い方のカット。前髪と後ろ髪の長さを合わせてグラデーションを消す。短く揃えることで、薄い部分と地毛の境界がぼやける」→ なぜそうするのかのロジックまで説明
  • 「U字に後退してきた方。残っている部分を活かしてトップにボリュームを持たせるカット」→ タイプごとに対応が違うことを示す

Threadsではテキストで考え方・ロジックを語り、Instagramではビフォーアフター写真＋簡潔な解説キャプションで展開する。

2番目：「薄毛カットの裏側」を見せる

美容師がどういう思考で薄毛のカットを設計しているかを開示する。これが「職人感」を生む。

POST EXAMPLES
  • 「薄毛カットで一番気を遣うのは『繋がり』。薄い部分と地毛の境目をどう自然に見せるか。いきなり短くするんじゃなくて、0.5mm刻みで変えていく」
  • 「初回のカウンセリングで必ず聞くこと。普段どっち分けか、仕事で帽子OKか、どのタイミングで一番気になるか。この3つで設計が全部変わる」
  • 「同じM字でも左右対称じゃない人がほとんど。片側だけ少し長めに残して、バランスを取ることがある」

3番目：「一から説明しなくていい安心感」を伝える

薄毛男性が美容室を変えられない最大の理由は「また説明するのがめんどくさい＋失敗が怖い」。この壁を壊すコンテンツ。

POST EXAMPLES
  • 「うちに来るお客さんの多くは、最初の一言が『実は薄毛のことを相談できる美容室を探してて…』。ここからはこっちのターン。カウンセリングで全部聞きます。次回からは何も言わなくても大丈夫な状態にします」
  • 「初回は30分かけてカウンセリング。薄毛のタイプ、進行の傾向、仕事環境、セットに使える時間。これを一回聞けば、次回からは『前回と同じで』でOK」
  • 「2回目以降のお客さんは、座った瞬間に前回からの変化を確認して、微調整の提案をこちらからする。聞かなくても察する」

4番目：「薄毛あるある」共感系

拡散＋入口として使う。笑いにするのではなく、「わかってる人が言ってる」感を出す。

POST EXAMPLES
  • 「薄毛が気になり始めてから、美容室選びのハードルが10倍になった人、たぶんめちゃくちゃ多い」
  • 「AGAの薬、調べれば調べるほど『飲み続けないといけない』が引っかかる。気持ちはわかる」
  • 「雨の日と風の日がこんなに怖くなるとは思わなかった。でもカットの設計次第で、風に強い髪型は作れる」
  • 「新しい美容室に行って、薄毛が余計に目立つカットにされた経験がある人、うちに来てほしい。同じ思いはさせない」

ビジネス構造との整合性

LTV（顧客生涯価値）が異常に高い

  • 月イチ来店 × 初回7,700円（リピートはもう少し安い想定）
  • 一度定着したら美容師を変えない → 年間10万前後 × 10年以上
  • 1人の新規獲得 ＝ 100万円以上のLTV
  • SNSで月に1人でも新規を獲得できれば、投資対効果は圧倒的に合う

新宿店オープン（2026年3月）との連動

  • 銀座店に加えて新宿店がオープン → キャパが倍になる
  • SNSで認知を広げる最も良いタイミング
  • 「新宿にも出せるぐらい需要がある」＝ 信頼の証拠としても使える
KEY POINT

リピート率が高いからこそ「初回の壁」を壊すことに全振りする。一度来れば通い続ける構造がある。SNSの役割は「初回来店の心理的ハードルを下げること」に集中。「この人なら大丈夫」「ここなら失敗しない」を投稿ごとに積み上げる。
