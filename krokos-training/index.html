Claude Code 導入講座

チャットAIは使える。でもGitHubやターミナルは知らない。そんな方のための、AIエージェントを安全に使いこなしてブランド業務を加速する実践講座。

SESSION 01

AIエージェントを知る

チャットAIとの違いを体感し、実際に最初の指示を出してみる。RPAとの比較、ツールの種類も学ぶ。

60分 座学 + 実践
SESSION 02

PCの仕組みを理解する

ターミナル、Homebrew、PATH。Claude Codeがどこで何をしているかを理解して、作業フォルダとAPIの仕組みを知る。

60分 座学 + ハンズオン
SESSION 03

ブランド業務に適用する

ルールファイルとMCPで、自社のトンマナやデータを活かす。メルマガのトンマナ分析やDMチラシ制作を実践。

60分 座学 + 実践ワーク
SESSION 04

セキュリティと応用設計

5つのリスクと対策、個人情報の扱い、Git。「次にAI化する業務」を自分で設計する力を身につける。

60分 座学 + 設計ワーク
