Claude Code 導入講座

チャットAIは使える。でもGitHubやターミナルは知らない。そんな方のための、AIエージェントを安全に使いこなす実践講座。

SESSION 01

AIエージェントって何？

チャットAIとの違いを体感する。RPAとの比較、ツールの種類を学び、「AIが手を動かす」感覚を掴む。

60分 座学 + デモ
SESSION 02

PCの裏側を知ろう

ターミナル、Homebrew、PATH。Claude Codeが「どこで何をしているのか」を理解して、作業フォルダとAPIの仕組みを知る。

60分 座学 + ハンズオン
SESSION 03

AIをカスタマイズする

ルールファイル（CLAUDE.md）とMCPの仕組みを学ぶ。AIは「学習」ではなく「マニュアルを読んでいる」ことを理解する。

60分 座学 + デモ
SESSION 04

安全に使って、チームで共有する

5つのセキュリティリスクと対策、Gitによるバージョン管理、GitHubでの共有方法。社内ルールを確認する。

60分 座学 + ワーク
