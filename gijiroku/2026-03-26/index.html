02 / 導線設計

3つの集客パターン

今回最も具体化した議論。既存のオーガニック運用に加え、広告を使った2つの新しい導線を並走させる設計。

① 既存IG リール・フィード → プロフィール → 無料動画講座 → 本講座

② IGブースト 広告でフォロワー増 → IGコンテンツで教育 → プロフィール → 無料動画講座 → 本講座

③ 動画広告 広告 → 直接ステップ動画講座 → オンライン講座 or 本講座
重要ポイント: ①と②を並走し、ROASを比較。③は予算配分を見ながら調整。久保さんの既存インスタ運用（マニュアル批判系の発信）は変えずに、当たっているものに広告をかけてブーストするイメージ。

動画講座のコンテンツ方針

  • マニュアル批判系が最も反響が高い（実証済み）
  • 知識提供よりもマインドセット・考え方の転換が刺さる
  • 「勉強した、やってみた、でも変わらないよね」への共感
  • 今までのインスタの焼き直し + 綺麗な場所で撮影し直すだけでOK

久保さんの発信スタイル

  • くびれ一辺倒より、幅広いトレーナー向け知識を発信
  • 「マニュアルと実践は違う」を言い続ける
  • たまにくびれ特化の引っかかりコンテンツ（例:「反り腰の方がいい」等）
  • コンテンツビジネス感を出しすぎず、等身大の良さを残す
03 / ターゲット分析

受講者の属性と行動パターン

久保さんとの雑談から、受講者の動機・属性・合格率に関する重要なインサイトが多数出た。

メインターゲット

デビュー後数年経ったトレーナーの「武器作り」。デビュー前の資格講座は多いが、デビュー後は少ない = 隙間市場

知識欲が高い層

合格率が高く、本講座向き。基礎知識の上積みがあるため習得が早い。少数だがLTVが高い

"時代遅れのマッチョ"層

数は多いが合格率は低い。勉強をサボってきた＋筋トレのポジティブホルモンで壁から逃げがち。基礎講座を挟む設計が必須

インサイト マッチョ層の行動メカニズム: 勉強で詰まる → ジムで筋トレ → ホルモンでポジティブになる → 詰まっていたことを忘れる → 知識が伸びない。ピラティスの台頭で「整える」トレンドに置いていかれて焦りを感じている層。広告で新規は取りやすいが、成約率は低い。

学びの動機パターン

売上・集客のため 独立トレーナーで崖っぷち。武器を手に入れて差別化したい 本講座向き
